Общие настройки сети

Все данные должны быть на одном языке, пустые элементы недопустимы, в тексте не должно быть HTML-тегов и управляющих символов.

Импорт XML

Загрузите готовый XML — форма заполнится автоматически. Если в файле несколько филиалов, данные откроются во вкладке массового ввода.

Данные станций

Основные поля по всем станциям

Минимально нужны: company-id, address, phone, working-time, rubric-id (можно оставить пустым — возьмётся из настроек сети).

company-id должен быть уникален для каждой АЗС и не меняться в следующих файлах.

company-id * Уникальный ID АЗС, латиница/цифры/-/_. address * Полный адрес одной строкой. phone * Формат: +7 (343) 375-13-99. working-time * Например: ежедневно 00:00-24:00. rubric-id * Цифровой ID рубрики. lon Долгота, точка как разделитель. lat Широта, точка как разделитель. name-other Уточнение/название АЗС. locality-name Город, если адрес неполный. копировать
